TicketKu

Dibuat untuk keperluan tugas kuliah. Program berjalan diatas docker dengan model distributed application dan distributed database, dan dapat diakses melalui 2 port. Terdapat "TicketKu app" dan "TicketKu Premium App" dimana pengguna pada Ticketku app dapat melakukan subscription pada pengguna di TicketKu premium app. Admin pada premium app dapat menerima atau menolak subscription request. Jika request untuk subscription telah diterima, maka pengguna pada user app dapat melihat segala konten berupa video dan gambar yang telah diupload oleh pengguna premium app yang disubscribenya. Program dibuat lengkap dengan sign in, register, error handling, pop up notification, token & cookies, authorization, caching dengan redis, dan CRUD database. Program ini dikembangkan dengan menggunakan react, javascript, php, bootstrap, mysql, postgreSQL, prisma, dan docker dengan menerapkan REST, dan JavaFx.

Aplikasi Jurnal

Aplikasi Journaling yang dibuat untuk memenuhi tugas mata kuliah Rekayasa Perangkat Lunak. Aplikasi ini memeiliki berbagai macam fitur seperti mencari artikel, mencatat jadwal, mencatat catatan perasaan, memantau grafik perasaan yang didapat berdasarkan data perasaan pengguna, dan lain-lain. Aplikasi ini dibuat dengan menggunakan tech stack: python 3, flet, SQLite, dan Matplotlib.

ChitGPT

Dibuat untuk keperluan tugas kuliah. Fitur yang diimplementasikan adalah Fitur pertanyaan teks (didapat dari database) dengan algoritma KMP dan BM, fitur kalkulator, fitur bertanya tanggal, fitur tambah dan hapus pertanyaan ke database dengan string matching. Pengguna dapat mengajukan pertanyaan dan program akan mencari jawaban tersebut berdasarkan pengetahuan yang ia sudah miliki. Program dibuat dengan typescript dan database postgreSQL dengan menggunakan prisma sebagai ORM.

e - Palancas

Dibuat atas permintaan sekolah untuk keperluan graduation semasa covid sehingga palancas/surat-menyurat antar murid terpaksa dilakukan secara virtual. Website responsive, terintegrasi dengan database data seluruh siswa SMA Santa Ursula BSD pada tahun 2021. Setiap siswa dapat menulis surat pada siswa lainnya baik secara anonymus maupun tidak. Terintegrasi dengan mailing sistem sehingga pada waktu yang ditentukan, setiap siswa akan mendapat email berisikan seluruh surat yang didapat dari siswa lainnya dalam bentuk pdf. Tech stack used: javascript, appscript, css, mongodb
