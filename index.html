TicketKu

Created for a college assignment, the program runs on Docker with a distributed application and distributed database model, accessible through 2 ports. There are two components: "TicketKu App" and "TicketKu Premium App." Users on the TicketKu App can subscribe to users on the TicketKu Premium App. Admins on the premium app can accept or reject subscription requests. If a subscription request is accepted, users on the user app can view all content, including videos and images, uploaded by the subscribed premium app users. The program includes features such as sign-in, register, error handling, pop-up notifications, token & cookies, authorization, caching with Redis, and CRUD database operations. It is developed using REACT, JavaScript, PHP, Bootstrap, MySQL, PostgreSQL, Prisma, and Docker, implementing REST and JavaFx.

ChitGPT

Created for a college assignment, the implemented features include a text question feature (retrieved from the database) using the KMP and BM algorithms, a calculator feature, a date inquiry feature, and a feature to add and delete questions to the database using string matching. Users can submit questions, and the program will search for answers based on the knowledge it possesses. The program is developed using TypeScript and a PostgreSQL database, with Prisma as the ORM.

e - Palancas

Developed at the request of the school for the graduation needs during COVID, where exchanges/letters between students had to be done virtually. A responsive website integrated with the database of all Santa Ursula BSD high school students in 2021. Each student can write letters to other students, either anonymously or not. Integrated with a mailing system so that at a specified time, each student will receive an email containing all the letters received from other students in PDF format. Tech stack used: JavaScript, AppScript, CSS, MongoDB.
